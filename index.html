Welcome

You have likely found this page because you scanned the QR code visible on a Space Invader-shaped magnet like the one shown in the picture below. On this page, you will find some information on the background of this project and all necessary files and instructions to create an InvaderMagnet yourself. How cool is that? ;)

How to create one

In order to create an InvaderMagnet, you will need access to a 3D printer. If you don't happen to own one, you can look for one in a local FabLab or Hackerspace, or you can have the model printed by an external service provider such as shapeways.com or sculpteo.com.

In the repository, you find several files for printing (you can download them all by clicking on the folders on top of the page):

  • InvaderMagnet.3dm is the model in Rhino's file format
  • InvaderMagnet.wrl is a plain file of the model without texture
  • InvaderMagnet.zpr is a textured version of the InvaderMagnet for printing on a Zprinter color 3D printer
  • QRcode.png is the image of the QR code that links to this site
  • QRtexture.png is an image with the QR code that fits the front of the model

Obviously, in order to print in color, you will need a color 3D printer, which can be quite hard to get your hands on. If you can only print one-colored, you need to add the QR code yourself. Just print it on a sticker or glue it to the invader.

In order to make the Invader magnetic, you will need to glue a small magnet (5mmx5mmx5mm) into the hole on the back of the model. We have used small neodymium magnets which worked quite well.

Background Idea

The InvaderMagnet was created by @sdraxler and @boden for the BarCamp in Siegen 2012 (http://barcamp-siegen.de/), as a small giveaway for our session on 3D printing. It was designed and printed at the HCI lab of the University of Siegen (http://www.uni-siegen.de/fb5/)

The idea was to create a small, useable object in order to show the potentials of the new technology. The QR code was added to provide a link for interested users to find out how such an object can be created.

The shape of a Space Invader was chosen because it was the logo of the Hackerspace Siegen (www.hasi.it) which cooperated with us for the occasion of the Barcamp.

The design is based on the great 8bit Space Invader Ornaments created by jglauche on Thingiverse (http://www.thingiverse.com/thing:5084), and licensed under the Attribution-Share Alike-Creative Commons license (CC BY-SA).
